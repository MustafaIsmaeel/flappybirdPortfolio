Flappy Bird — Portfolio Build

A compact, responsive Flappy Bird remake built with HTML5 Canvas — polished glass UI + neon highlights. Play, inspect the code, and fork the project.

Score: 0

Controls: Space / Click / Tap to flap. Press D to toggle hitboxes.

Features: Responsive canvas scaling, local highscore, WebAudio cues, debug mode.

Project Info

  • Engine: Vanilla JavaScript (Canvas)
  • Assets: SVG/PNG + WAV (included placeholders)
  • Deploy: GitHub Pages (enable Pages → Branch: main, folder: /)
